# Apps - POLOMIMIN

Thư mục chứa 7 modules ứng dụng của POLOMIMIN:

| Module | Mô tả |
|--------|--------|
| `_shared/` | Components dùng chung |
| `academy/` | Học viện - Khóa học AI |
| `ai/` | AI Center - Chat, Image, Video |
| `erp/` | Quản lý doanh nghiệp |
| `hr/` | Quản lý nhân sự |
| `mes/` | Quản lý sản xuất |
| `sales/` | Quản lý bán hàng |
| `wms/` | Quản lý kho |

Mỗi module có thể truy cập qua: `apps/{module}/index.html`
